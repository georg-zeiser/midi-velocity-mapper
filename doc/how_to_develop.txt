instructions to develop/debug on a windows 10 machine

make sure, you have all the required tools installed (Visual Studio 2019, Visual Studio Code, nodejs, angular)

backend:
    * open ./MidiVelocityMapper.sln in "Visual Studio 2019"
    * run
    -> browser window should open with the url http://localhost:5000/api

webclient:
    * open folder ./WebClient in "Visual Studio Code"
    * start a terminal console, make sure your working directory is "WebClient"
    * npm install -f
    * npm run-script start
    -> browser window should open with url http://localhost:4200
